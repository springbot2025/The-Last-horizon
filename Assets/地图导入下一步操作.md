# 地图导入 - 下一步操作步骤

## ✅ 已完成

## 🎯 接下来的步骤

### 步骤1：配置图片设置（重要！）

#### 操作：
```
1. Project窗口 → 点击选中 地图1.png

2. Inspector窗口（右侧）查看图片设置

3. 必须修改的设置：
   ┌─────────────────────────────────┐
   │ Texture Type: Default ▼        │ ← 选择 Default
   │                                 │
   │ Read/Write Enabled: ☐           │ ← 必须勾选 ✓
   │                                 │
   │ [Apply]                         │ ← 点击应用
   └─────────────────────────────────┘
```

**验证：**
- [ ] Texture Type = Default
- [ ] Read/Write Enabled = 已勾选 ✓
- [ ] 已点击 Apply 按钮

---

### 步骤2：添加MapImageLoader脚本（如果还没有）

#### 检查是否需要添加：
```
1. Hierarchy窗口 → 选中 MapManager 对象
2. Inspector窗口 → 查看是否有 MapImageLoader 组件
```

#### 如果没有，添加组件：
```
1. Hierarchy窗口 → 选中 MapManager对象

2. Inspector窗口 → 点击 "Add Component" 按钮

3. 搜索框输入：MapImageLoader

4. 点击 "MapImageLoader" 添加
```

**验证：**
- [ ] Inspector窗口能看到 MapImageLoader 组件

---

### 步骤3：分配地图图片到脚本

#### 操作：
```
1. Hierarchy窗口 → 确认 MapManager 已选中

2. Inspector窗口 → 找到 MapImageLoader 组件

3. 找到 "Map Image" 属性（第一个属性）

4. 从 Project窗口拖动 地图1.png 到这个槽中
```

**操作示意：**
```
Project窗口              Inspector窗口
┌──────────────────┐    ┌──────────────────┐
│ 📷 地图1.png     │──→ │ Map Image        │
│                  │    │ ┌──────────────┐ │
│                  │    │ │ [None]       │ │ ← 拖到这里
│                  │    │ └──────────────┘ │
└──────────────────┘    └──────────────────┘
```

**验证：**
- [ ] Map Image槽中显示 "地图1"
- [ ] 不再显示 "[None]"

---

### 步骤4：配置识别参数（可选）

#### 如果知道地图信息，可以设置：
```
MapImageLoader组件中：

Cell Size In Pixels: [96]    ← 如果方格是96像素，填96
                                不知道填0（自动检测）

Map Width: [0]               ← 如果知道列数，填写
                                不知道填0（自动检测）

Map Height: [0]              ← 如果知道行数，填写
                                不知道填0（自动检测）

Verbose Logging: ☑          ← 建议勾选（显示详细日志）
```

---

### 步骤5：运行图片识别

#### 方法A：使用右键菜单（推荐）
```
1. Inspector窗口 → MapImageLoader组件

2. 点击组件右上角的三个点（⋮）图标

3. 在下拉菜单中选择："从图片加载地图"

4. 系统会立即执行识别
```

#### 方法B：运行场景
```
1. 点击Unity顶部工具栏的 Play 按钮 ▶️

2. Map脚本会自动检查并加载图片数据
   （需先使用方法A运行一次识别）
```

**推荐使用方法A**，可以在编辑模式下运行。

---

## ❓ 如果遇到问题

### 问题1：Read/Write Enabled无法勾选
```
解决：
1. 先将 Texture Type 改为 Default
2. 等待Unity处理
3. 然后就可以勾选 Read/Write Enabled 了
```

### 问题2：找不到MapManager对象
```
解决：
1. Hierarchy窗口 → 查找 MapManager
2. 如果没有，创建一个空的GameObject
3. 命名为 MapManager
4. 添加 Map 和 MapImageLoader 组件
```

### 问题3：识别失败或结果不对
```
解决：
1. 检查Console窗口的错误信息
2. 确认Read/Write Enabled已勾选
3. 检查地图图片是否有清晰的网格线
4. 尝试调整Border Color Threshold参数
```
