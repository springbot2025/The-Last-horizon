# 从图片导入地图 - 快速操作指南
**Unity 2022.3.62 专用**

## ⚡ 5分钟快速版

### 步骤1：导入图片（1分钟）
```
Unity Project窗口 → 拖入地图PNG文件
```

### 步骤2：配置图片（1分钟）
```
选中图片 → Inspector窗口：
✓ Read/Write Enabled = 勾选（重要！）
✓ Texture Type = Default
点击 Apply
```

### 步骤3：添加脚本（1分钟）
```
Hierarchy → 选中 MapManager对象
Inspector → Add Component → MapImageLoader
```

### 步骤4：分配图片（30秒）
```
Inspector → MapImageLoader组件
Map Image槽 → 拖入地图PNG
```

### 步骤5：运行识别（30秒）
```
Inspector → MapImageLoader组件 → 右键菜单
选择 "从图片加载地图"
```

### 步骤6：查看结果（1分钟）
```
Console窗口 → 查看加载信息
Play按钮 → 查看地图显示
```

---

## 📋 关键设置

### 必须勾选！
```
图片导入设置 → Read/Write Enabled ✓
```
**如果不勾选，脚本无法读取像素数据！**

### 参数设置（如果知道）
```
Cell Size In Pixels: 96（如果方格是96像素）
Map Width: 列数（如果知道）
Map Height: 行数（如果知道）
```
**如果不知道，保持0，会自动检测**

---

## 🎯 核心操作

### 右键运行识别
```
Inspector窗口 → MapImageLoader组件 → 点击三个点（⋮）
→ 选择 "从图片加载地图"
```

### 或者运行场景
```
点击 Play 按钮
Map脚本会自动读取加载的数据
```

## ❗ 常见问题

### Read/Write Enabled是灰色？
```
解决：Texture Type 改为 Default
```

### 识别不到网格线？
```
1. 检查网格线颜色是否够深
2. 降低 Border Color Threshold（如0.2）
3. 确认Read/Write Enabled已勾选
```

### 地图显示不对？
```
1. 查看Console错误信息
2. 检查图片是否正确分配
3. 确认MapImageLoader和Map脚本在同一个对象
```
