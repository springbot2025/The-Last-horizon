# 导入地图图片 - 详细步骤

## 📍 您的文件位置
```
Windows下载文件夹：地图1.png
目标：导入到Unity项目中
```

## 🚀 操作步骤

### 方法一：直接拖拽（最简单，推荐）⭐⭐⭐

#### 步骤1：找到Unity的Project窗口
```
1. 打开Unity 2022.3.62
2. 打开您的项目
3. 查看Unity编辑器左下角的 Project 窗口
   （如果看不到，按 Ctrl+9 或 Window → General → Project）
```

#### 步骤2：打开Windows文件资源管理器
```
1. 按 Win+E 打开文件资源管理器
2. 导航到下载文件夹：
   - 通常路径：C:\Users\您的用户名\Downloads
   - 或者在文件资源管理器地址栏输入：%USERPROFILE%\Downloads
```

#### 步骤3：拖拽文件到Unity
```
1. 在文件资源管理器中找到 地图1.png
2. 点击并按住鼠标左键（不要松开）
3. 拖动到Unity编辑器的 Project 窗口
4. 拖到 Assets 文件夹区域（通常是Project窗口的根目录）
5. 松开鼠标

Unity会自动导入这个文件
```

**操作示意：**
```
文件资源管理器          Unity Project窗口
┌──────────────────┐    ┌──────────────────┐
│ 📷 地图1.png     │──→ │ Assets           │
│                  │    │  ├─ Scenes       │
│                  │    │  ├─ Scripts      │
│                  │    │  └─ ...          │
│                  │    │                  │
│                  │    │  ← 拖到这里！   │
└──────────────────┘    └──────────────────┘
```

### 方法二：右键导入

#### 步骤1：在Unity Project窗口中操作
```
1. Unity Project窗口 → 右键点击 Assets 文件夹
   （或右键点击Project窗口的空白区域）
2. 选择 "Import New Asset..."
```

#### 步骤2：选择文件
```
1. 文件选择对话框会打开
2. 导航到下载文件夹：
   - 左侧：快速访问 → 下载
   - 或地址栏输入：C:\Users\您的用户名\Downloads
3. 找到并选中 地图1.png
4. 点击 "Import" 按钮
```

---

## ⚙️ 配置图片设置（重要！）

导入后，必须配置图片设置才能使用：

### 步骤1：选中图片
```
Project窗口 → 点击刚导入的 地图1.png
```

### 步骤2：配置Inspector设置
```
Inspector窗口（右侧）会显示图片导入设置：

找到并修改以下设置：
┌─────────────────────────────────┐
│ Texture Type: Default ▼         │ ← 选择 Default
│                                 │
│ Read/Write Enabled: ☐           │ ← 必须勾选！
│                                 │
│ Max Size: 2048 ▼               │
│ Filter Mode: Bilinear ▼        │
│ ...                            │
│                                 │
│ [Apply]                         │ ← 点击应用
└─────────────────────────────────┘
```

### 关键设置说明：

1. **Texture Type = Default**
   - 点击下拉菜单，选择 "Default"
   - 这是必需的，否则无法勾选 Read/Write Enabled

2. **Read/Write Enabled = ✓ 勾选**
   - 这个必须勾选！
   - 如果不勾选，MapImageLoader无法读取图片像素数据

3. **点击 Apply 按钮**
   - 应用所有设置
   - Unity会重新处理图片

**验证：**
- [ ] Texture Type = Default
- [ ] Read/Write Enabled = 已勾选 ✓
- [ ] 已点击 Apply 按钮

## 📝 快速操作总结

**最简单的方法（3步）：**
```
1. 打开文件资源管理器 → 找到 地图1.png
2. 拖动 → Unity Project窗口的Assets区域
3. 选中图片 → Inspector → Read/Write Enabled ✓ → Apply
```

**完成！** 图片已导入并配置完成。

## ❓ 常见问题

### 问题1：拖拽没反应
```
解决：
1. 确保拖到Project窗口的Assets区域（不是标题栏）
2. 确保Unity窗口处于活动状态
3. 等待Unity编译完成（查看底部状态栏）
```

### 问题2：Read/Write Enabled是灰色
```
解决：
1. 先将 Texture Type 改为 Default
2. 等待Unity处理
3. 然后就可以勾选 Read/Write Enabled 了
```

### 问题3：找不到下载文件夹
```
Windows快捷键：
- Win+R → 输入：%USERPROFILE%\Downloads → 回车

或在文件资源管理器：
- 左侧：快速访问 → 下载
```
