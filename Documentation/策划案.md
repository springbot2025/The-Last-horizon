#  策划案

## **世界观**

 这是一个有着魔法和高科技的世界。在这里，你可以看到魔法师从高性能压缩魔力机械中放出跳动的火舌，也能看到战士手上的枪弹缠绕着魔力的光束。在魔力和科技的加持下，人类快速地发展。

但这也是一个荒芜的世界，魔法与科技确实给人类带来了高速的发展，但也带来了一个问题——资源的分配，带来了，战争。因为战争，人类的数量已经被削减至不足一亿。资源被破坏殆尽，目光所及最多的就是黄沙和弹片。

即便如此，掌权者们依旧没有放弃杀伐。但战争已经将人类折磨得太久了，为了不让自己以后掌控的大地空无一人，那些掌权者不得不在讨论后达成一个协议——这个世界将被分为两个大联合国，而每个联合国又由多个成员国组成，一方叫寰宇联合国，他们主张在胜利后将人类最后的资源用于太空航行科技开发，让最后的人类前往其他国家生活。另一方叫地球联合国，他们主张用最后的资源开发自然生态修复加速科技，尽快让地球回到资源充盈的状态并带领人类继续在地球上生存。而这最后也是决定人类命运的战争将在这两个联合国之间开始。

寰宇联合国成员国：灵国，宙斯帝国，赫斯合众国

地球联合国成员国：墨丘集团，赫拉共和国，莱特第三共和国

虽然这个世界的国家被分为了两个大联合国，但也有很多人，他们或不相信曾做出让人民置于战场之上的掌权者，想自己找一条解决战争的道路，或不想人生被如此随意分配，总之，他们离开了自己原本的国家，决定自己去活自己的人生，他们之中有的人选择独行，有的人选择与同伴一起，但更多的人选择了一个地方，那个地方被人称为"避风港"。传说，避风港里有很多离开了自己国家的人，他们没有组织性，没有纪律，但是每当避风港的首领"神之言"有命令，避风港内没有人能拒绝。但经过地球联合国缜密调查后并未调查出此事的缘由和避风港的其余情报，甚至连该组织的据地都未调查清楚。

但是，这个世界又真的有那么单纯吗？也许，掌权者的争权并未结束；也许，各个联合国的目的没有那么单纯；也许，这个世界并不止三方势力。但这一切都好像与你无关，因为你只不过是一个战场上的无名之人，只是为了自己或自己的国家全力以赴罢了。

 

## **游戏规则**

·游戏开始时，双方各操控两个角色，初始位置在各自出生点。

·两方随机决定先手，每回合中，行动顺序为：先手1号角色→后手1号角色→后手二号角色→先手二号角色

·每次行动可选择攻击/移动/恢复护盾/使用主动技能等操作，无法越过障碍，（特殊说明除外）

·对局中，只能在相邻方格间移动，不能走斜线

·对局中，每个主动技能仅能使用一次，复活后重置该次数。

·对局中，角色生命值降为0或以下视作死亡，死亡后可消耗复活机会在出生点复活，若无复活机会，则该角色移出对局。复活后的下一回合无法行动。

胜利条件：己方角色进入对手基地 或 对手在场上不存在角色。

## **地图设定**

目前设置固定数量地图，便于平衡游戏（组长正在设计中）

方格设置：

空地：无效果

战壕：在其中只能被相邻方格中的人锁定，但不影响自身攻击距离

障碍：位于方格边缘，无法穿过其进行攻击/移动/释放技能等操作

污染水域：无法进入，但可以越过其进行攻击/释放技能（如果距离足够）

传送点：移动到该格后可消耗一次行动次数移动到对应的另一格

## **角色设定**

生命值：受到伤害即减少

护盾：受到伤害即减少（优先级大于生命值）。每回合行动时可选择护盾+1

移动距离：一次移动的距离

复活次数：默认1次

主动技能：可主动发起，每局游戏仅能使用一次，复活后重置该次数。

被动技能：满足条件后即生效

 

·钢腕

生命值：3

护盾：1

阵营：地球联合国

能力：魔力手盾：一回合未被攻击后可展开（主动），可抵挡一次受到的伤害。

背景：墨丘集团和赫拉共和国共同生产的第三代改造人，也是第一代自主意识型改造人，他们拥有自主思考的能力，能够像第二代改造人一样同时使用科技与魔力，一般作为战争的先遣队出场。

背景：有人说他是"被遗弃之人"，有人说他是"知晓一切之人"。
